    CRIE!

    Crie playlists com viagens musicais entre diversos gêneros.

    Seu objetivo é criar playlists com diferentes paisagens musicais. Cada paisagem é um conjunto de músicas que tem o mesmo gênero musical. Entre duas paisagens musicais temos uma música de transição, que é uma música que combina os gêneros das duas paisagens musicais.

    Fácil

    Escolhendo primeiro a música de transição, ou seja, uma música que misture dois gêneros musicais, fica mais fácil identificar e selecionar as músicas das paisagens musicais.

    Médio

    Escolhendo primeiro as músicas da primeira paisagem musical é preciso identificar uma música que contenha tanto o gênero desta paisagem quanto um outro gênero musical diferenciado, para assim escolher as músicas da segunda paisagem musical.

    Difícil

    Escolhendo primeiro dois gêneros musicais diferentes e selecionando as músicas para cada paisagem musical, você precisa ser bastante eclético para identificar então uma música que misture estes dois gêneros, está será sua música de transição.
